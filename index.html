PAGE DESIGN LOGIC — 頁面設計邏輯
ZONE 01 — TOP
購買決策區（最上方）
集中所有促使下單的資訊——價格、促銷碼、信任保障、多角度圖片，讓訪客不需捲動就能完成決策。
ZONE 02 — BELOW FOLD
快速摘要（Feature / Contents）
緊接其後，用條列格式讓訪客秒懂「這是什麼、盒子裡有什麼」，並提供說明書 PDF 下載，降低售後疑問。
ZONE 03 — NAV
錨點導覽列
充當目錄，方便技術型買家直接跳到 Specs，不干擾一般用戶的閱讀動線。
ZONE 04–06 — MAIN BODY
Overview → Features → Specs
遵循「情感說服 → 功能論述 → 技術驗證」的遞進邏輯，先讓人想要，再讓人相信，最後提供數據支撐。
ZONE 07–09 — BOTTOM
技術支援 + 評價 + 相關商品
放在頁尾，服務「已被說服但還有疑慮」的用戶，同時帶動交叉銷售。
